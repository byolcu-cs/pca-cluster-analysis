# PCA and Cluster Analysis

Principal component analysis and clustering project focused on dimensionality reduction, pattern discovery, and segment interpretation.

## Overview
This project applies principal component analysis (PCA) and clustering techniques to identify meaningful structure in multivariate data. The analysis focuses on reducing dimensionality, interpreting principal components, and comparing clustering results to identify distinct groups within the dataset.

## Methods
- Principal Component Analysis (PCA)
- Scree plot / component selection
- Hierarchical clustering
- K-means clustering
- Silhouette-based cluster evaluation
